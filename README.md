🐍 Python 90 Days Bootcamp

Welcome to my Python 90 Days Bootcamp repository.
This repository documents my daily Python learning journey, including exercises, mini-projects, and coding challenges completed during a structured 90-day Python training program*.

The goal of this project is to build strong Python programming skills through consistent daily practice and real coding projects.


📚 Topics Covered

During this bootcamp, I practice and build projects around:

* Python fundamentals
* Variables and data types
* String manipulation
* Conditional statements (`if`, `elif`, `else`)
* Loops (`for`, `while`)
* Functions and modules
* Lists, dictionaries, and tuples
* File handling
* Error handling
* Mini Python projects
